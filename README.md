# :moneybag: CLUB CHAIN

> CLUB CHAIN은 블록체인 기반 회비 관리를 바탕으로한 동호회 커뮤니티 서비스입니다.

- 진행 기간 : 21.08.23 ~ 21.10.08

- 목표 
  - 원하는 동호회에 가입하고 투명한 회비 관리로 건강한 동호회 운영을 할 수 있습니다.
  - 다양한 사람들과 취미를 공유할 수 있는 동호회 커뮤니티입니다.
